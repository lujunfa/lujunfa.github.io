我的第一个独立的静态博客

发表于 2018-03-25 | 分类于 Blogs

如何建立自己独立的静态博客

说明：在互联网的大时代，很多社区和门户网站都向用户提供了博客的功能，使用户有了向大家分享自己的所想所得的机会，但这些博客的设计风格都是千篇一律，大致相同，用户不能根据自己的喜好进行定制的，并且用户博客空间的域名也是这些社区网站的二级域名，这让人多多少少有种寄人篱下的感觉。于是，作为一个对技术有追求的我，为了不受制这种限制，我采用了将hexo生成的博客站点托管于coding、github第三方平台，阿里云域名解析重定向到相关托管平台的page演示地址，从而实现静态博客站点。

###需要准备的工具
+因为hexo是基于node.js的一个博客站点开发工具，所以需要先安装node.js环境，然后再安装hexo工具：安装hexo的命令npm install hexo-cli -g

+安装git bash

###开始
1、github和coding创建项目
2、hexo创建站点
hexo init <folder> <folder> 自己随意命名 cd <folder> 进入到已命名的<folder>目录中 npm install
3、

1、新建一个about页面,命令如下：
hexo new page “about”
